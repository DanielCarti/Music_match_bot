# Как разместить Telegram-бота на PythonAnywhere (2026 Edition)

Это руководство поможет вам запустить бота 24/7.

### Шаг 1: Подготовка на PythonAnywhere

1.  **Создайте аккаунт:** На [PythonAnywhere](https://www.pythonanywhere.com/). Для доступа к Яндекс.Музыке и VK на бесплатном тарифе могут быть ограничения (прокси), но для платных тарифов всё работает идеально.
2.  **Подтвердите email.**

### Шаг 2: Загрузка файлов

1.  Перейдите во вкладку **"Files"**.
2.  Создайте папку проекта, например `music_bot`.
3.  Загрузите в неё файлы из вашего проекта:
    *   `main.py`
    *   `requirements.txt`
    *   `.env` (создайте его на сервере вручную через кнопку "New file", чтобы не пушить ключи в интернет).

### Шаг 3: Настройка окружения

1.  Откройте **Bash Console** во вкладке "Consoles".
2.  Создайте виртуальное окружение:
    ```bash
    mkvirtualenv bot-venv --python=/usr/bin/python3.10
    ```
3.  Установите библиотеки:
    ```bash
    cd music_bot
    pip install -r requirements.txt
    ```

### Шаг 4: Настройка ключей (.env)

В папке `music_bot` создайте файл с названием **`.env`** и впишите туда свои токены:
```env
TELEGRAM_BOT_TOKEN=ваш_токен
VK_ACCESS_TOKEN=ваш_токен_vk
SPOTIPY_CLIENT_ID=ваш_id (необязательно)
SPOTIPY_CLIENT_SECRET=ваш_секрет (необязательно)
```
*Наш бот автоматически подхватит эти данные благодаря библиотеке `python-dotenv`.*

### Шаг 5: Запуск 24/7 (Always-on)

1.  Перейдите во вкладку **"Tasks"**.
2.  В разделе **"Always-on tasks"** введите полную команду запуска:
    ```bash
    /home/ВАШ_НИКНЕЙМ/.virtualenvs/bot-venv/bin/python /home/ВАШ_НИКНЕЙМ/music_bot/main.py
    ```
    *(Замените ВАШ_НИКНЕЙМ на логин в PythonAnywhere).*
3.  Нажмите **"Add task"**.

### Шаг 6: Контроль

Следите за работой через иконку **"Log"** в разделе Tasks. Если там появится "Bot started...", значит всё отлично!

---
**Примечание по Spotify:** Бот работает с плейлистами Spotify через файлы TXT/CSV, поэтому дополнительные настройки API для пользователей не требуются.
